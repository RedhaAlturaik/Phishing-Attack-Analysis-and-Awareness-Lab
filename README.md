# Phishing Attack Analysis and Awareness Lab

## Overview

This project was conducted as part of a university cybersecurity course to study phishing attacks and social engineering techniques in a controlled laboratory environment. The objective was to understand how phishing campaigns operate, evaluate their effectiveness, and identify methods for detecting and preventing such attacks.

## Objectives

* Understand the principles of phishing attacks.
* Analyze the role of social engineering in cybersecurity incidents.
* Study how attackers attempt to obtain sensitive information.
* Explore techniques used to identify and mitigate phishing threats.
* Increase awareness of common phishing indicators.

## Tools Used

* Kali Linux
* BlackEye (for educational demonstration purposes)
* Ngrok
* Social Engineering Toolkit (SET)

## Methodology

The project examined the workflow of a phishing attack in a controlled lab environment. The study included:

1. Creating a simulated phishing scenario.
2. Analyzing how phishing websites imitate legitimate services.
3. Observing how social engineering techniques influence user behavior.
4. Evaluating methods for detecting phishing attempts.
5. Reviewing security practices that help prevent credential theft.

## Key Findings

* Phishing attacks rely heavily on user trust and social engineering.
* Users are more likely to interact with messages that appear legitimate.
* Security awareness training significantly reduces phishing success rates.
* Multi-factor authentication can mitigate the impact of compromised credentials.

## Prevention and Mitigation

* Verify URLs before entering credentials.
* Enable multi-factor authentication (MFA).
* Avoid clicking suspicious links in emails or messages.
* Use email filtering and anti-phishing solutions.
* Conduct regular cybersecurity awareness training.

## Learning Outcomes

Through this project, we gained practical knowledge of:

* Social engineering techniques
* Phishing attack methodologies
* Security awareness and user education
* Cybersecurity risk assessment
* Ethical considerations in security testing
